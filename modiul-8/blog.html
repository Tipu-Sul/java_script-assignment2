"var", "let", "const" both of them are keyword in Java Script by which we can declare variables name
var : var is a functional scope variables declaration. var can be accessed in the same function from any scope. Globally declared var can be accessed any where of code and allows redeclaration of variables.
let let is a block scope variable declaration .It can be accessed from only the same scope. it allow reassign but not redeclare.
const const is also a block scope variables declaration.It not allow reassign.Generally const are use to declare variables, which are no need to change like array, object.

The use case of null and undefinded:
Null : If we declare a variable but don't assign any value, then if we want to print the value of this variable, then we get null.
undefinded : If we try to print a variable but we never declare it in the scope or code block then the result come as undefinded.
